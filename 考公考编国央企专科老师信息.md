# **考公（计算机专业特别版）**

## **考公岗位类型（非常全面！！）**

### **行政类岗位**

- **综合管理类**：办公室文秘、行政管理、政策研究等岗位
- **财务会计类**：预算管理、会计核算、审计监督等岗位
- **法律事务类**：法制办、司法局、法院书记员等岗位
- **🔥信息技术类**：网络维护、软件开发、数据管理等岗位

### **执法类岗位**

- **公安系统**：人民警察、交警、刑警、🔥网警等
- **市场监管**：食品药品监督、质量监督、价格监督等
- **城管执法**：城市管理、环境监察、建筑监管等
- **税务稽查**：税务检查、反偷漏税、税收征管等

### **专业技术类岗位**

- **卫生健康类**：疾控中心、卫生监督、医疗机构管理等
- **教育文化类**：教育局、文化局、体育局等行政岗位
- **农林水利类**：农业技术推广、林业管理、水利工程等
- **交通运输类**：交通规划、道路管理、运输监管等

### **🔥计算机专业热门岗位**

- **网络安全岗**：网络安全监管、信息安全审计、网络犯罪侦查
- **数据分析岗**：政务数据分析、统计信息处理、决策支持
- **系统维护岗**：政府信息系统维护、数据库管理、软件运维
- **数字化建设岗**：电子政务开发、智慧城市建设、信息化规划

## **各类考试及时间节点**

### **国家公务员考试**

- **报名时间**：每年10月中下旬（2024年为10月15日-24日）
- **笔试时间**：每年11月底或12月初（2024年为11月26日）
- **面试时间**：次年2-3月
- **录用时间**：次年6-7月

### **省级公务员考试**

- **联考时间**：每年3月下旬或4月上旬笔试

- 单独考试

  ：

  - 北京：每年11-12月
  - 上海：每年11-12月
  - 江苏：每年3月
  - 浙江：每年5月

### **选调生考试**

- **定向选调**：每年10月-次年1月
- **普通选调**：每年3-4月
- **急需紧缺选调**：不定期发布

### **🔥计算机专业专项考试**

- **证监会计算机类**：考试时间120分钟，满分100分，包含证券期货基础知识和专业知识两部分
- **银保监会信息科技类**：专门设置计算机相关专业科目
- **税务局信息技术岗**：侧重税务信息化和数据处理
- **公安机关网络安全岗**：重点考查网络安全和信息技术

- **文化系统**：图书馆、博物馆、文化馆等
- **🔥科研院所**：各类研究机构技术人员、**信息化建设**
- **农林水利**：农技推广、林业、水利等
- **🔥新闻出版**：广播电视、报社、出版社**数字化运营**等

## **考试时间节点**

### **教师招聘**

- **春季招聘**：3-5月（主要时间段）
- **秋季招聘**：8-10月（部分地区）
- **寒假招聘**：12月-次年2月（少数地区）

### **医疗卫生招聘**

- **上半年**：3-6月
- **下半年**：9-11月
- **不定期招聘**：根据需要随时发布

## **🔥计算机专业求职建议**

### **技术准备**

1. **编程语言**：熟练掌握Java、Python、C++中的2-3种
2. **数据结构与算法**：LeetCode刷题500+，重点掌握动态规划、图论（没学）
3. **数据库**：MySQL、Oracle、Redis等主流数据库
4. **框架技术**：Spring、MyBatis、Vue.js等企业级框架（spring还好多没学，java太多没学了）
5. **云计算**：了解阿里云、腾讯云、AWS等云服务

### **项目经验**

1. **完整项目**：至少2个完整的项目经验（前后端+数据库）
2. **技术栈**：项目涵盖微服务、分布式、高并发等技术（高并发，协程编程 异步编程，线程gil锁，用协程多进程）
3. **团队合作**：体现团队协作和项目管理能力
4. **开源贡献**：GitHub活跃度，参与开源项目

### **证书加分**

1. **软考证书**：系统分析师、系统架构设计师、网络工程师
2. **厂商认证**：华为HCIE、思科CCIE、阿里云ACE
3. **行业认证**：PMP项目管理、CISSP信息安全
4. **英语能力**：CET-6或托福雅思成绩

### **面试准备**

1. **技术面试**：算法题、系统设计、技术深度
2. **项目介绍**：STAR法则描述项目经验
3. **行业了解**：目标企业的业务模式和技术栈
4. **软技能**：沟通表达、团队合作、学习能力
